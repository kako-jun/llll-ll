畑の8畝を  
ぜんぶカラッカの種とカモミールの種でかいがいに埋めたのに

5つ目まで収穫して  
クリーエの種を1つもゲットできなかったわけです

すべてを収穫した時  
ゲットできたのは（狙ってもない）サンレモンの種２つでした

えぇ～～～  
なにこれ……

サクナヒメより難しくない？

ガーデニングソイルでの交雑率が分からないわけです

ザナラーンソイルG3の交雑率が90%であることは  
山のように記事が見つかるわけですが

その中の誰も  
ガーデニングソイルでの交雑率を調べてない

数十記事あっても  
ほとんどが同じような内容

無個性か

シュラウドソイルの場合  
使わなくても収穫数が0になるってことはないから

同じように  
ザナラーンソイルは  
元々ある0%でない交雑率に下駄を履かせるに過ぎないと予想してた

下駄を履かせた結果  
90%になるのなら

下駄を履く前は50%くらいだと予想してた

でも感覚的に  
10%以下な気がする

ひょっとしたら  
ガーデニングソイルでの交雑率は0%なのかもしれない

交雑に関して  
ググって出てくるFF14の記事の多くは

筆者がすでに交雑に詳しすぎるのか  
本人にしか分からない表記が多くて

客観性を失ってて  
他人に説明する文章になってないわけです

例えば

カラッカ x カモミール = クリーエの種

みたいな記述

x記号で  
隣り合った畝であることを意味するのは分かりやすいけど

ちゃんと「の種」って書かないと

カラッカの野菜を植えるのか？  
って初心者なら思うだろう

まぁ……  
これは植えようとしても植えられないから  
1回目で気づくので良しとして……

厳密に書いたら

カラッカの種 x カモミールの種 = カラッカの野菜 + カモミール + （〇〇%の確率でクリーエの種）

になっちゃう

カラッカの種でカラッカの野菜が  
カモミールの種でカモミールができるのは当たり前すぎるので

そこを略すのは全然イイ！

ただ  
クリーエの種をゲットできるのは

カラッカの野菜を収穫した時なのか？  
カモミールを収穫した時なのか？

その両方なのか？

そこをちゃんと書いてほしいッ！

クーリエの種とか  
書いてる場合もあるし

実は書いてる本人が  
初心者以上に理解ってないんじゃあないか？

畝を2つ使って交雑に挑戦した場合

種は  
畝1で1つ  
畝2で1つ

の計2つをゲットできる可能性がある？

期待値の最大は2個で  
最小が0個ってこと？

カラッカ x カモミール = クリーエの種  
という表現方法では

クリーエの種が  
カラッカのほうの畝から採れるのか？  
カモミールのほうの畝から採れるのか？

表現ができない

カモミール x カラッカ = クリーエの種

のように  
入れ替えて書くしかない  
（左が畝1、右が畝2を表すという、暗黙の了解）

仮に

\- そのような区別は意味ない  
\- 畝1と畝2はまったく等価である  
\- 種は両親の組み合わせだけで決まり、畝1と畝2のどちらにも同じ種ができうる

という仕様ならば  
その情報こそが1番大事で

初心者に対して真っ先に説明すべき前提知識だから

何番煎じか分からない組み合わせの表を書くくらいなら

それを書けばいいのに……  
と思うわけです

もっとひどいのだと

ホウキグサ x アーモンド = ホウキグサ

みたいな記述も見たことある

ループ交雑について教えたいのだと思うけど……

ホウキグサの種を植えて  
ホウキグサを収穫できるのは当たり前だろうッ！？

ホウキグサの種 x アーモンドの種 = ホウキグサ + アーモンド + （〇〇%の確率でホウキグサの種）

になる  
珍しい組み合わせだからループできるんじゃあないのか

それならば

略さず  
左辺と右辺を合わせようッ！

数式を模してるのに  
気持ち悪くならないのか……？

そして  
この場合に（運良く）ホウキグサの種をゲットできる畝は

親である  
ホウキグサの種を植えた畝なのだろうけど

アーモンドの種を植えたほうの畝でも  
ゲットできる可能性があるのか皆無なのかくらい書いてほしいッ！

FF14の攻略記事には

カタカナ、英語の羅列とか  
一見さんお断りの符牒が頻出するものですが

それでも  
極やスキル回しや家具とかの攻略なら  
読めばちゃんと理解できる文章が揃ってるのです

交雑だけが

なぜか  
日本語力が壊滅した攻略記事ばかりな印象です

本人的には書いてるつもりの前提知識が多すぎて  
ハイコンテクストすぎるってやつです

読む側が想像で補完しても  
その想像が正しい保証はないので

たとえ交雑の組み合わせが正しかったとしても  
ノイズにしかなってないわけです

たぶん  
「ああ、なんか急に交雑を解説したくなったぞー！」  
って人が現れるたび

既存のダメダメな解説を  
「なるほど、こういうふうに書く文化があるんだな！」と  
違和感持たずに真似して書いて

似たような記事の  
量産を繰り返した結果として

うちわ向けな内容に偏っていったのでしょう

まるで……  
交雑関連の記事自体が

ダメな交雑を  
数世代繰り返した結果という感じ……

しかも終わりのない  
ループ交雑になってる気がしたので

クリーエの種が手に入らない  
八つ当たり先として

既存記事への不満を  
5分で書いてみました

種は  
マーケットボードで買ったら負けだと思ってる！
