このポートフォリオは長らく Next.js で動いていた。問題なく動いてはいたが、ほぼ変化しないページを描くために、訪問のたび React ランタイムを配っていた。「作ったものを並べるだけ」のサイトには、少し重すぎた。

そこで **Zola** に移した。Rust 製の静的サイトジェネレータで、バイナリ1つ、`node_modules` なし、フルビルドが1秒かからない。出てくるのは素の HTML と CSS で、JavaScript は働きどころ——アプリのフィルタ・訪問カウンタ・日替わり絵——にだけ顔を出す。

### 自分のテーマをドッグフードする

配色のベースは、自作の Zola テーマ **avel**。llll-ll は avel のテンプレを直に継承してはいない——ポータルは btop 調の自前レイアウトだ——が、黒×緑の美学とビルドの流れは共有している。自分のサイトを自分のテーマで建てるのが、avel の粗を見つける一番の近道だ。

### 変えなかったところ

- ワンページ。アプリを見るのに画面遷移しない
- モバイルファースト、PC は中央寄せ
- 4言語（en / ja / zh / es）、それぞれを対等な URL に

移行はフェーズで進めた。土台 → i18n → 操作パネル群 → そしてこれ、blog。この記事のような投稿が最後のピースだ。作りながら、作っているものを書き留める場所。

> 静的なバイナリ1つで足りるなら、私は静的なバイナリを配りたい。
