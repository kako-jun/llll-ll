这个作品集很长一段时间跑在 Next.js 上。它能用，但为了渲染一个几乎不变的页面，每次访问都要发一份 React 运行时。对于一个"把做过的东西列出来"的站点，这有点太重了。

于是我把它迁到了 **Zola**——一个用 Rust 写的静态站点生成器。单个二进制，没有 `node_modules`，整站构建用时远不到一秒。它产出的是纯 HTML 和 CSS；JavaScript 只在真正需要的地方出现——应用筛选、访问计数、每日一图。

### 用自己的主题吃自己的狗粮

配色基底来自我自己写的 Zola 主题 **avel**。llll-ll 并不直接继承 avel 的模板——门户是 btop 风格的定制布局——但黑配绿的美学与构建流程是共享的。用自己的主题搭自己的站，是发现 avel 毛刺最快的办法。

### 保持不变的部分

- 单页。浏览应用不切换页面
- 移动优先，桌面居中
- 四种语言（en / ja / zh / es），每种都是一等公民的 URL

迁移分阶段进行：脚手架 → 国际化 → 交互面板 → 然后就是这个，blog。像这篇一样的文章是最后一块拼图：一个边做边记录的地方。

> 如果一个静态二进制就能搞定，我宁愿发那个静态二进制。
